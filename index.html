Set up your Trips app
This takes about 10 minutes and you only do it once. After setup your trips sync between your phone and laptop, and still work offline.
1
Put the app online — GitHub (free)
a. Go to github.com → sign up free if needed.

b. Click + → New repository. Name it trips. Set to Public. Click Create repository.

c. Click "uploading an existing file" → drag in your 3 files (index.html, sw.js, manifest.json) → Commit changes.

d. Settings → Pages → Source: Deploy from branch → main → / (root) → Save.

✅ Live at https://YOUR-USERNAME.github.io/trips/ in ~60s
2
Create your cloud database — Supabase (free)
a. supabase.com → sign up → New project. Any name, any region, set a password → Create. Wait ~2 min.

b. SQL Editor in sidebar → tap box below to copy → paste → click Run:
-- Run this once in Supabase SQL Editor CREATE TABLE app_data ( key TEXT PRIMARY KEY, value TEXT NOT NULL, updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE app_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "open" ON app_data FOR ALL USING (true) WITH CHECK (true);

c. Storage → New bucket → name trip-media → turn on Public bucket → Create. Then Policies → New policy → trip-media → Full customization → name it public, check ALL operations, expression true → Save. (This lets photos and files sync between devices.)

d. Project Settings → API → paste your Project URL and anon key below:
3
Install on your phone
iPhone: Open your GitHub Pages URL in Safari → Share icon → Add to Home Screen → Add.

Android: Chrome → three-dot menu → Add to Home Screen.
